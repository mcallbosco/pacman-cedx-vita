How to install on your Vita

Clicking Build Vita installer produces two files: a ~2 MB pacmancedx.vpk (loader + LiveArea) and a ~130 MB pacmancedx-data.zip (game data). Keeping the data outside the VPK avoids VitaShell installing hundreds of megabytes through the package installer; you copy the loose data over the network instead.

  1. On your computer, unzip pacmancedx-data.zip. You should get one top-level folder named pacmancedx/ with the game files inside.
  2. On the Vita, start FTP (for example VitaShell: press Select for the FTP server). From your PC, connect once with an FTP client and upload that entire pacmancedx folder into ux0:data/ so the Vita ends up with ux0:data/pacmancedx/…
  3. Still on that connection (or over USB if you prefer), copy pacmancedx.vpk onto the Vita as well — for example into ux0:data/. Then in VitaShell highlight the VPK, press Triangle → Install, and accept the extended permissions prompt.
  4. Launch PAC-MAN CE DX from your LiveArea.

Runs entirely in-browser via fflate.
